1. 信息收集与使用

我们不收集、不存储、不传输任何个人数据。 本应用是一款完全在本地运行的桌面美化工具，用于管理桌面图标的自动隐藏与显示动画。

本应用不会要求您创建账户，不会访问您的文件、联系人、位置、浏览器历史记录或任何其他敏感信息。 所有配置（例如您选择的动画模式、蒙版透明度、省电开关偏好）仅保存在您设备的本地配置文件（settings.ini）中，永远不会上传至任何服务器。

2. 权限与功能说明

为实现图标自动隐藏、省电适配等核心功能，本应用会调用以下系统接口。所有操作均在本地完成，不会将数据传出设备：

  • 前台窗口检测：监测桌面图标区域是否正在进行重命名或弹出右键菜单，以决定是否保持图标可见。本功能仅识别窗口类名（如 Edit 控件），不记录任何用户输入的内容。
  • 开机自启动：使用 Windows 应用商店的 StartupTask API 提供开机自启选项，您可在应用内菜单随时开启或关闭。该功能仅让应用随系统启动，不涉及任何数据收集。
  • 动态壁纸兼容：通过标准的窗口枚举判断是否运行了第三方动态壁纸软件（如 Wallpaper Engine），以便正确渲染桌面背景。不会访问壁纸软件内的任何个人文件或数据。

3. 第三方服务

本应用不使用任何第三方分析工具、广告 SDK 或遥测服务。 我们不会与任何第三方共享任何信息，因为我们根本没有收集任何信息。

由于本应用不在本地存储之外处理任何个人数据，因此不存在数据泄露、滥用或未授权访问的风险。 您的动画偏好设置仅以明文形式保存在您当前用户的 AppData 文件夹中，其他应用程序无法访问。

AutoICON Privacy Policy

Effective Date: May 10, 2026
Last Updated: May 10, 2026

AutoICON ("the Application") takes your privacy seriously. This Privacy Policy is intended to clearly and transparently explain how the Application handles information.

1. Information Collection & Use

We do not collect, store, or transmit any personal data. This Application is a desktop enhancement tool that operates entirely locally, managing the automatic hiding and showing animations of desktop icons.

The Application does not require you to create an account, nor does it access your files, contacts, location, browsing history, or any other sensitive information. All configuration preferences (such as your chosen animation mode, mask opacity, and power-saver toggle) are stored solely in a local configuration file (settings.ini) on your device and are never uploaded to any server.

2. Permissions & Functionality

To implement core features such as automatic icon hiding and power-saving adaptation, the Application invokes the following system interfaces. All operations are performed locally and no data is transmitted off-device:

  • System Power Status: Reads battery status and power plans (e.g., "Power Saver" mode) solely to determine whether to reduce animation frame rates to conserve energy. No personally identifiable information is read.
  • Foreground Window Detection: Monitors whether the desktop icon area is currently being renamed or has a context menu open, in order to decide whether to keep icons visible. This feature only identifies window class names (such as the Edit control) and does not record any user-inputted content.
  • Auto-start at Boot: Uses the Windows Application StartupTask API to provide an auto-start option, which you can enable or disable at any time from the in-app menu. This feature simply allows the Application to launch with the system and involves no data collection.
  • Live Wallpaper Compatibility: Uses standard window enumeration to detect whether third-party live wallpaper software (such as Wallpaper Engine) is running, in order to correctly render the desktop background. No personal files or data within wallpaper software are accessed.

3. Third-Party Services

The Application does not use any third-party analytics tools, advertising SDKs, or telemetry services. We do not share any information with any third parties because we simply do not collect any information in the first place.

4. Data Security

Because the Application does not process any personal data beyond local storage, there is no risk of data leakage, misuse, or unauthorized access. Your animation preference settings are stored only in plain text within your current user's AppData folder and are inaccessible to other applications.

5. Children's Privacy

The Application does not collect personal data from anyone, including children under the age of 13. We do not knowingly collect or retain personal information from children.

6. Privacy Policy Updates

We may update this Privacy Policy from time to time. Any updated policy will be made available to you via the app store listing or through an in-app notice. We encourage you to review this policy periodically to stay informed of any changes.
